# Offline PDF Assets

Place offline PDF files for tools in this folder.

## Guide to Safe Scouting

Expected filename for the tool page:

- `guide-to-safe-scouting.pdf`

The `tools/guide-to-safe-scouting` page will automatically open this file if present.

Suggested source:

- https://www.scouting.org/health-and-safety/gss/
- Legacy PDF URL often used by units: https://filestore.scouting.org/filestore/pdf/34416.pdf

If the PDF is not present, the tool page shows fallback instructions and the official online link.
